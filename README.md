## kinect班
スペースシャトルの操縦
### やること1
* kinectから手先の高さを測定
* 右（左）手をあげると右（左）に旋回（回転+並進）
* 手を挙げていないときは直進
### やること2
* 手を開けているときはビームを出す

## leap班

* ロケットをマウス追従して動かす

* マウス右ボタンをドラッグしたサイズ・位置の惑星を配置する
* HP/HIT数を表示　10回当たると終了
* リプレイボタンを設置
